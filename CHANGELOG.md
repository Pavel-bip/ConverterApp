# Changelog

Все заметные изменения в проекте будут задокументированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/), и проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.0] — 2026-04-24

### Добавлено
- Экран конвертации валют с выбором исходной и целевой валюты из выпадающих списков.
- Поддержка быстрого пересчёта по последнему использованному курсу.
- Возможность добавления текстовой заметки к операции конвертации.
- Экран управления валютами: добавление, редактирование курса и удаление валют.
- Загрузка актуальных курсов валют из API Центрального банка РФ.
- Экран истории операций с поиском по дате, коду валюты и заметке.
- Экспорт истории операций в формат CSV.
- Экран аналитики с построением линейного графика динамики курса выбранной валюты.
- Отображение статистики частоты использования валют.
- Нижняя панель навигации между четырьмя основными экранами.
- Локальное хранение данных с использованием базы данных Room (SQLite).
- Обработка ошибок ввода и сетевых запросов с выводом информативных сообщений.

### Исправлено
- Некорректное форматирование результата конвертации при вводе суммы с несколькими знаками после запятой - результат всегда отображается с двумя знаками.
